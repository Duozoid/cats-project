PARDOFELIS

The line of the Borneo cat includes a single genus with the geological name of marbled cat (Pardofelis). It is very small, so for a long time it was thought that it was related to small cats - but no, to large cats (Panthers). The so-called Catopuma also belong to the Borneo cats. Some consider them marbled cats, some single them out into a separate genus - their position is unclear. The Catopuma include the Borneo cat (after which the line is named) and the Temminck cat.
